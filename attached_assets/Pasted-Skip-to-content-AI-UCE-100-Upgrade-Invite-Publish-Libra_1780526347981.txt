Grammar Inference — extracts generative rules from structured text/code/docs. Stores rules + parameters, not the text itself.
Program Synthesis — finds the shortest program that reproduces the file. For highly structured data this is extreme (sine wave → 120 bytes).
Holographic Codebook — for high-entropy (random/encrypted) data. XORs blocks against a pre-shared 500MB random codebook so only an address index is stored.
